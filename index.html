Javascript Cheetsheet

Iterators

Function Return new array Description
.forEach() No Calls a change for each element if an array
.map() Yes Create a new array by performing a function on each array
.filter() Yes Create a new array with elements that pass a test
.reduce() No Runs a funtion on each array value to produce a single value
.reduceRight() No The same a .reduce() but runs right to left
.every() No Check if every element passes a test
.some() No Check if some array values pass a test
.indexOf() No Check an array for a element value and returns the position of the first occourence of this value
.lastIndexOf No The same as .indexOf() but returns the position of the last occourence of this value
